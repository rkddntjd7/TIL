# 자바 연산식

- 연산이란? 
  - 데이터를 처리하여 결과를 산출하는 것.

- 연산자 : 연산에 사용되는 표시나 기호
- 피연산자 : 연산 대상이 되는 데이터(리터럴, 변수)
- 연산식 : 연산자와 피연산자를 이용하여 연산의 과정을 기술한 것
