# mySQL 명령어

1. mysql 접속 : mysql -u아이디 -p 엔터 -> 비밀번호 입력
2. show databases; -> db의 리스트를 표시하라.
3. use world; -> world db를 사용하겠다.
4. show tables; -> world db의 테이블리스트를 표시하라.
5. desc city; -> city 테이블의 구조를 표시하라.
6. select * from city; -> city 테이블의 내용을 보여달라.
